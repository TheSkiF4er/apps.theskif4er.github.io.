Сайт готов к запуску.

Как открыть:
1. Распакуйте архив.
2. Откройте файл index.html в браузере.
3. Пароль по умолчанию: fpv58

Что умеет сайт:
- Открывает таблицу только после ввода пароля.
- Числовые ячейки можно отмечать кликом.
- Выбранные ячейки становятся красными.
- Заголовки Band / CH и названия строк не нажимаются.
- Отметки сохраняются в браузере.

Как изменить пароль:
- Откройте файл script.js
- Найдите строку:
  const SITE_PASSWORD = 'fpv58';
- Замените значение на свой пароль.

Важно:
Это простая защита на стороне браузера. Для полноценной защиты нужен серверный бэкенд.
