Нужна первичная настройка

Сайт готов, осталось подключить базу

В архиве уже лежат файлы config.js, config.example.js и supabase-schema.sql. Заполните URL и Anon Key вашего проекта Supabase, затем откройте сайт через локальный сервер или любой статический хостинг.

  1. Создайте бесплатный проект Supabase.
  2. Выполните SQL из supabase-schema.sql.
  3. Вставьте URL и Anon Key в config.js.
  4. При желании смените ключи через SHA-256 и обновите значения в supabase-schema.sql.
